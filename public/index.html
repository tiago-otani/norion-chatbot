Norion Chatbot — página de teste

Esta página simula o site norion.ind.br para testar o widget de chat (canto inferior direito).

Como embutir no site oficial:

Adicione esta linha no HTML do site (head ou body):

<script src="https://SEU-APP.up.railway.app/widget.js" defer></script>

Substitua SEU-APP.up.railway.app pelo domínio gerado no Railway.
